# Server-Performance-Stats-

Script sederhana untuk melihat performa server:
- CPU usage
- Memory usage
- Disk usage
- Top 5 proses berdasarkan CPU & Memori

Cara menjalankan:
```bash
chmod +x server-stats.sh
./server-stats.sh

```
